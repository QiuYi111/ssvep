🔬 独立游戏与数字疗法 Web 技术栈调研报告

针对项目： 《星空与萤火》SSVEP 冥想应用
核心技术挑战： 锁定 120/60 FPS 极速渲染（确保 SSVEP 频点精准）、海量粒子物理运动、极低延迟音频生成、前端直接读取蓝牙脑电信号。

一、 核心视觉渲染引擎 (Visual & Rendering)

传统的 DOM 渲染（CSS 动画）在处理几十个元素时就会出现微小的卡顿。对于 SSVEP，我们需要直接调用显卡（GPU）性能，以下是业界标配：

1. PixiJS (极力推荐的 2D 王者)

适用场景： 星空、萤火虫粒子、2D 迷雾、光晕扩散。

为什么选它： 它是目前地球上最快的 Web 2D 渲染引擎（基于 WebGL）。你可以在屏幕上同时渲染 10,000 只发光的萤火虫，并且让它们按照柏林噪声（Perlin Noise）轨迹游荡，MacBook Pro 依然能稳如老狗地跑满 120 帧。

AI 友好度： 极高。你可以直接让 AI：“用 PixiJS 写一个粒子发射器，模拟 15Hz 闪烁的萤火虫聚集效果”。

2. Three.js / React Three Fiber (R3F) (3D 降维打击)

适用场景： 立体星球、3D 莲花绽放、体积光（God Rays）、空间深度感知。

为什么选它： 如果你想让“冥想环境”有纵深感（比如从平面的星星变成穿越宇宙的 3D 视野），Three.js 是不二之选。React Three Fiber 则是将 Three.js 完美融入 React 的神器，极其优雅。

高阶玩法 (Shader 着色器)： 冥想 APP 里那种“柔和流动的极光”或“水面波纹”，用图片是做不出来的，都是用 GLSL 着色器（Shaders）算出来的。AI 极其擅长写 Shader 效果。

二、 神经诱导与动态音频 (Audio-Visual Entrainment)

我们不仅需要播放背景音乐，还需要根据脑电波的专注度实时改变音频的频率和质感，这属于 Web Audio API 的范畴。

1. Tone.js (动态生成音频与脑波同步)

核心能力： 它不是用来播放 MP3 的，它是一个在浏览器里运行的模块化合成器。

SSVEP 绝配： 我们在 GDD 中提到的“双脑同步音波（Binaural Beats）”，用 Tone.js 只需要几行代码就能实时生成。你可以让左耳播放 400Hz，右耳播放 415Hz，从而在用户大脑中诱导出 15Hz 的拍频（与视觉上的萤火虫 15Hz 形成完美的跨模态共振）。当用户走神时，你可以用代码让音频加上一个低通滤波器（Low-pass filter），产生声音“沉入水底”的惩罚反馈。

2. Howler.js (空间音效与环境音)

核心能力： 业界标准的声音播放库。

适用场景： 处理雨声、篝火声、风声等环境白噪音，支持 3D 空间音效（比如让流星划过的声音从屏幕左边“飞”到右边）。极具沉浸感。

三、 脑电信号连接与处理 (Hardware & Brain-Computer Interface)

既然是纯 Web 端（甚至不用套外壳），如何让浏览器读取头环的数据？

1. Web Bluetooth API (浏览器原生蓝牙)

彻底改变游戏规则： Chrome 和 Edge 原生支持。不需要用户下载任何底层驱动或驱动助手。用户打开网页，点击“连接头环”，浏览器直接通过蓝牙低功耗 (BLE) 接收来自脑电头环的原始 EEG 数据流。

优势： 即点即玩，极大地降低了用户的使用门槛（这也是为什么推荐你在 Mac + Chrome 上开发的原因）。

2. BrainFlow (WebAssembly 版) / eeglib (信号处理)

核心能力： 读取到脑电波只是一堆杂乱的数字，需要进行 FFT（快速傅里叶变换）或 TRCA（任务相关成分分析）来提取 15Hz 或 20Hz 的能量。

WebAssembly 的威力： BrainFlow 是顶级的 BCI 库，它支持编译为 WebAssembly（Wasm）在浏览器中以接近 C++ 的速度运行，能在极低延迟（<100ms）内算出用户的专注度指数。

四、 动效与界面编排 (Animation & Choreography)

1. GSAP (GreenSock Animation Platform)

核心能力： 业界最强的网页动画库，获奖的交互网站几乎 100% 都在用它。

适用场景： 游戏关卡之间的转场、UI 元素的优雅浮现、复杂的连环动画（例如：专注度达标 -> 触发萤火虫聚拢 -> 延迟 1 秒 -> 星光连线 -> 爆发光晕）。GSAP 擅长处理这种如同“电影导演”般的精确时间轴编排。

2. Framer Motion

核心能力： React 生态中最优雅的物理动画库。

适用场景： 适合用来做冥想 APP 中丝滑的菜单切换、弹簧物理效果（Spring physics），能让整个应用充满“苹果级别的质感”。
